Manual de usuario

Este manual de usuario es una guía para utilizar el sistema de camaras corporativas de Cammer Corporativo Industrial S.A. de S.V

Para utilizar el sistema de camaras corporativas de Cammer Corporativo Industrial S.A. de S.V, debe seguir los siguientes pasos:

  1. Primero, debe registrarse en el sistema con su usuario y contraseña.
  2. Una vez registrado, puede acceder a la lista de camaras corporativas disponiblesen el sistema.
  3. Para subir datos a la camara corporativa, debe seleccionar la camara corporativa que desee subir datos y luego seleccionar el tipo de dato que desee subir.
  4. Una vez seleccionado el tipo de dato, puede subir el archivo de datos seleccionado.
  5. Para visualizar los datos subidos a la camara corporativa, debe seleccionar la camara corporativa que desee visualizar y luego seleccionar el tipo de dato que desee visualizar.
  6. Una vez seleccionado el tipo de dato, puede visualizar los datos subidos a la camara corporativa.
